# Restro

A simple restaurant management web application built with PHP and MySQL.  
This project allows you to manage food items, categories, and orders through an easy-to-use admin panel.

## Features

- **Food Management**
  - Add, update, and delete food items
  - Upload and manage food images
  - Set featured and active status

- **Category Management**
  - Create and edit categories for food items
  - Manage active/inactive categories

- **Admin Panel**
  - Secure login for administrators
  - Dashboard to view food and categories
  - Role-based access control (if implemented)

- **Responsive UI**
  - Built with Bootstrap for mobile-friendly interface
